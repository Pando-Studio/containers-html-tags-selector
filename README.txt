With CHTAGS, you'll be able to choose your own HTML tags for each containers in your pages. This plugin is an extension for WPBarkery Page Builder.

== Description ==

Thanks to Containers HTML Tags Selector (CHTAGS), you are now able to choose what HTML tags you want for each 
containers in your pages. Currently, with WPBakery Page Builder, you can build your pages with rows and columns,
but you can't choose if you're gonna create a simple div or a new footer, and this is a real issue for SEO. This
problem is now resolved, with this new plugin, you'll increase your SEO without any effort.
This plugin only work as an extension of WPBakery Page Builder. This plugin doesn't work with theme or theme-child who
already personalizes templates of rows and columns of WPBakery Page Builder.

== Installation ==

1. Upload the first plugin "WPBarkery Page Builder" to the `/wp-content/plugins/` directory
2. Upload the second plugin : "Containers HTML Tags Selector" to the `/wp-content/plugins/` directory
3. Check if the current theme has no files named 'vc_row.php', 'vc_row_inner.php', 'vc_column.php', 'vc_column_inner.php'
4. Activate the first plugin through the 'Plugins' menu in WordPress
5. Activate the second plugin through the 'Plugins' menu in WordPress
